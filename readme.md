## 背景

试想以下场景：

- 当一边吃着零食一边看电影的时候，**不方便触碰鼠标**，但是需要进行暂停/快进等操作
- 需要在一定距离外使用蓝牙鼠标操作

现在你可以使用**Vision Mouse**来处理上述的场景！

## 功能

**Vision Mouse**使用[mediapipe](https://developers.google.com/mediapipe/solutions/vision)来提供手部关键点的识别，提供了鼠标的所有行为控制：

- 移动
- 点击
- 右键
- 拖拽
- 滚动
